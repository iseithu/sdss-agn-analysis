# SDSS AGN Spectral Analysis

This project presents an analysis of Active Galactic Nuclei (AGN) using SDSS DR18 data.

- Data selection using SQL queries
- AGN subclass filtering
- Cross-matching using SDSS CrossID
- Spectral visualization
- FITS file analysis using Python

## Files

- `data/` → CSV datasets
- `spectra/` → plotted spectra images
- `fits/` → raw spectral FITS files
- `notebook/` → Jupyter Notebook analysis

## Tools Used

- Python (Astropy, NumPy, Matplotlib)
- SDSS SkyServer
- Jupyter Notebook
